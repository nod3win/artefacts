\begin{exo}[Le suffixe de chaque fonction]
Donne le suffixe utilisé pour nommer une molécule dont la fonction principale est :
\begin{enumerate}[label=\alph*)]
  \item un acide carboxylique
  \item un aldéhyde
  \item une cétone
  \item un alcool
  \item une amine
\end{enumerate}
\end{exo}

\begin{exo}[Qui est prioritaire ?]
Pour chaque paire de fonctions, indique laquelle est la fonction principale (la plus prioritaire).
\begin{enumerate}[label=\alph*)]
  \item acide carboxylique / alcool
  \item cétone / amine
  \item alcool / alcène
  \item aldéhyde / éther
\end{enumerate}
\end{exo}

\begin{exo}[Lire un nom, retrouver la fonction principale]
D'après son suffixe, quelle est la fonction principale de la molécule nommée ?
\begin{enumerate}[label=\alph*)]
  \item acide propanoïque
  \item butan-2-one
  \item pentan-1-ol
  \item propanal
\end{enumerate}
\end{exo}

\begin{exo}[Nommer une molécule à une seule fonction]
Donne le nom systématique de chaque molécule.
\begin{enumerate}[label=\alph*)]
  \item \ce{CH3-CH2-CHO}
  \item \ce{CH3-CH2-CH2-OH}
  \item \ce{CH3-CO-CH2-CH3}
\end{enumerate}
\end{exo}

\begin{exo}[Où commence la numérotation ?]
Quel numéro porte le carbone de la fonction dans la chaîne principale ?
\begin{enumerate}[label=\alph*)]
  \item le carbone du $-\ce{COOH}$ d'un acide carboxylique
  \item le carbone du $-\ce{CHO}$ d'un aldéhyde
\end{enumerate}
\end{exo}

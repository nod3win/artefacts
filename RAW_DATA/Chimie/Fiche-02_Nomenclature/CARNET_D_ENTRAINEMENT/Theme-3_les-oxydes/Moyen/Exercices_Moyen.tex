\begin{exo}[nommer un oxyde de métal à valence variable]
Le métal a plusieurs valences : détermine-la ($\ce{O}=-\mathrm{II}$, molécule neutre) et indique-la en
chiffres romains.
\begin{enumerate}[label=\alph*)]
  \item \ce{FeO}
  \item \ce{Fe2O3}
  \item \ce{Cu2O}
  \item \ce{CuO}
\end{enumerate}
\end{exo}

\begin{exo}[écrire un oxyde de métal à valence variable]
La mention (valence) fixe la charge du métal ; croise-la avec \ce{O^2-}.
\begin{enumerate}[label=\alph*)]
  \item oxyde de fer(II)
  \item oxyde de cuivre(II)
  \item oxyde de mercure(II)
  \item oxyde de plomb(IV)
\end{enumerate}
\end{exo}

\begin{exo}[nommer un oxyde non métallique \ce{X2O_n}]
L'élément $X$ est présent \emph{deux} fois : le rapport O/X vaut $n/2$. Donne le préfixe fractionnaire et
le nom.
\begin{enumerate}[label=\alph*)]
  \item \ce{N2O3}
  \item \ce{N2O5}
  \item \ce{Cl2O7}
\end{enumerate}
\end{exo}

\begin{exo}[écrire un oxyde non métallique fractionnaire]
Le préfixe donne le rapport O/X ; il y a deux atomes de $X$. Écris la formule.
\begin{enumerate}[label=\alph*)]
  \item sesquioxyde de phosphore
  \item hémipentoxyde de phosphore
  \item hémioxyde de chlore
\end{enumerate}
\end{exo}

\begin{exo}[une même paire d'éléments, plusieurs rapports]
Le chlore forme plusieurs oxydes. Pour chacun, calcule le rapport O/Cl et donne le nom.
\begin{enumerate}[label=\alph*)]
  \item \ce{Cl2O}
  \item \ce{Cl2O3}
  \item \ce{Cl2O5}
\end{enumerate}
\end{exo}
